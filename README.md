# Microservicio de Proyectos


## Pasos

1. Instalar dependencias - "npm install"
2. Configurar variables de entorno teniendo como ejemplo a "env.example"
3. Levantar imagen de docker con base de datos de postgreSQL - "docker compose up -d"


## Recomendaciones

1. Puedes utilizar "DBeaver" para visualizar la base de datos, te conectas con las credenciales del "env.example"
